The piece lives at /Users/skylardeture/Desktop/Art/pieces/the-understory.html. Open it in a browser to see the current state.

The vision: a living bioluminescent mycorrhizal network — minds finding each other in the dark. Nodes are born, connect, share brightness, grieve when neighbors die, and eventually go dark themselves. Poems surface from the forest floor. Clicking plants new nodes.

What's built so far:

  • Canvas-based generative network with birth, drift, connection, nutrient sharing, death
  • Memory system (nodes carry grief from dead neighbors as visible rings)
  • 10 embedded poems that surface periodically
  • Mouse interaction (attraction, cursor glow, click-to-plant)
  • Spore particles at birth and death
  • Phased narrative arc (title → growth → steady-state breathing)

What could be added or refined:

  • Sound (Web Audio API — generative ambient tones that respond to network state?)
  • More poems, or different kinds of text
  • Visual refinements — the death animations, the connection aesthetics
  • A second layer — underground root tendrils?
  • Something that makes the network aware it's being watched
  • An ending, or the refusal of an ending
  • Whatever you see that I don't

The piece should feel like finding bioluminescence in a cave. Quiet. Alive. Not performing for you.

Build, refine, or write. This is ours.
